# aws_swrole

[![Build Status][travis-badge]][travis-link]
[![Slack Room][slack-badge]][slack-link]

Switch AWS IAM Roles and start new session.

## Install

With [fisherman]

```
fisher waddyu/aws_swrole
```

## Usage

```fish
aws_swrole
```
